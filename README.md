# Juego Vida Universitaria

Proyecto desarrollado en Unity para la materia Desarrollo de Contenidos Digitales y Multimediales.

## Descripción
Juego interactivo en 2D donde el jugador toma decisiones sobre situaciones de la vida universitaria.

Cada decisión afecta variables como:

- Promedio académico
- Dinero
- Estrés
- Energía
- Relaciones

## Tecnologías utilizadas
